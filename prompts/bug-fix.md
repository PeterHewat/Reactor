# Bug Fix

Analyze and fix this bug systematically:

- **Root Cause Analysis**: Identify the underlying cause, not just symptoms
- **Reproduction**: Create minimal steps to reproduce the issue
- **Impact Assessment**: Determine scope and severity of the bug
- **Fix Strategy**: Choose the most appropriate solution approach
- **Testing**: Verify the fix resolves the issue without introducing regressions
- **Edge Cases**: Ensure the fix handles related edge cases
- **Type Safety**: Address any TypeScript errors or unsafe patterns
- **Error Handling**: Improve error handling if the bug relates to unhandled errors
- **Performance**: Ensure the fix doesn't introduce performance regressions
- **Documentation**: Update docs if the bug revealed gaps in understanding

Provide:

1. Clear explanation of what was wrong
2. The implemented fix with reasoning
3. Test cases to prevent regression
4. Any related improvements or cleanup
5. Verification steps to confirm the fix works

Follow debugging best practices:

- Use debugger and logging strategically
- Check browser dev tools and network tab
- Verify data flow from Convex to UI
- Test across different browsers and devices
